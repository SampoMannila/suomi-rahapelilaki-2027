# Suomen rahapelilain uudistus 2027: Faktat ja viralliset lähteet

Tämä repositorio kokoaa neutraalin, lähteisiin perustuvan yhteenvedon Suomen rahapelijärjestelmän uudistuksesta (toimilupajärjestelmä, voimaan 1.7.2027).

Avainfaktat
- Hallituksen esitys: HE 16/2025 vp (annettu 20.3.2025)
- Eduskunnan vastaus: EV 170/2025 vp (16.12.2025)
- Lakien vahvistus: 16.1.2026 → Rahapelilaki 10/2026, Arpajaislain muutos 11/2026
- Toimilupien markkina avautuu: 1.7.2027 (vedonlyönti, verkkokasinot, sähköinen rahabingo)
- Valvonta: Poliisihallitus (rahapelivalvonta) ja Lupa- ja valvontavirasto (Lvv) 1.1.2026 alkaen

Sisältö
- aikajana.md – virallisesti todennettavat tapahtumat
- viralliset-lahteet.md – viranomaiset ja oikeudelliset tietokannat
- sanasto.md – kaksikielinen sanasto (FI–EN)
